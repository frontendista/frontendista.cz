## 1. Local development

### 1.1 Prerequisites

1. Install node.js (>16)
2. Install pnpm
3. Configure your hosts file to map `127.0.0.1 frontendista.test`

### 1.2 Environment variables

You can also check the [.env.example](./.env.example) file.

### 1.3 Installation

```sh
# Clone repository
git clone git@github.com:frontendista/frontendista.cz.git
cd frontendista.cz

# Configure your .env file then continue

# Install dependencies
pnpm install

# Start development server
pnpm dev
```
