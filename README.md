# Frontendista.cz

Personal website of Pavel Sušický

## 1. Local development

### 1.1 Prerequisites

-   [**Node.js**](https://nodejs.org)
-   [**pnpm**](https://pnpm.io)

### 1.2 Installation

```sh
git clone git@github.com:frontendista/frontendista.cz.git

cd frontendista.cz

pnpm install
```
